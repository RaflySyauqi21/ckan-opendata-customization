## Feature: Data Request Button
This feature adds a public data request button to the CKAN header.

### Purpose
- Provide an official channel for public data requests
- Integrate CKAN with external request workflows
- Reduce manual communication overhead

### Integration
- Google Form (managed by DISKOMINFO)
