## ENGLISH ##

# UI Theme Customization
Interface customizations are made by modifying the `main.css` file on the CKAN-based Open Data portal.

## Modified Elements
- Header and top navigation
- Navbar hover menu
- Primary system color
- Login button
- Dashboard text color
- Data search area
- Popular tags
- Footer

## Purpose of Changes
- Adjust the agency's visual identity
- Improve interface consistency
- Clarify navigation elements and user interactions

# UI Theme Customization
Penyesuaian antarmuka dilakukan melalui modifikasi file `main.css`
pada portal Open Data berbasis CKAN.

## Elemen yang Dimodifikasi
- Header dan top navigation
- Hover menu navbar
- Warna primer sistem
- Tombol login
- Warna teks dashboard
- Area pencarian data
- Tag populer
- Footer

## Tujuan Perubahan
- Menyesuaikan identitas visual instansi
- Meningkatkan konsistensi tampilan antarmuka
- Memperjelas elemen navigasi dan interaksi pengguna
