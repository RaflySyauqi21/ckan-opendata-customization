## Feature: Navigation Optimization
This feature redirects navbar links (Datasets, Organizations, Groups)
to user dashboards instead of public listings.

### Purpose
- Prevent OPD users from accessing global dataset views
- Encourage users to manage their own data
- Reduce input mistakes across organizations

## Fitur: Optimasi Navigasi
Fitur ini mengalihkan tautan navbar (Dataset, Organisasi, Grup)
ke dasbor pengguna, bukan ke daftar publik.

### Tujuan
- Mencegah pengguna OPD mengakses tampilan dataset global
- Mendorong pengguna untuk mengelola data mereka sendiri
- Mengurangi kesalahan input di seluruh organisasi
