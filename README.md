## ENGLISH ##

# CKAN Open Data Customization
This internship mini-project involves developing and customizing the CKAN-based Bangkalan Regency Open Data portal.

## Scope of Work
- Interface (UI) adjustments
- Dataset visibility settings
- User navigation optimization
- Adding a data request feature

## Technology
- CKAN
- Python
- HTML
- CSS
- Git

## Notes
Changes are made directly to the CKAN core without using extension plugins.

## Folder Structure
- `patches/` : Patch files documenting CKAN core modifications
- `docs/`    : Feature explanations and screenshots
- `ui/`      : UI-related configuration reference

## How to Apply Patch (Optional)
Patches can be applied manually or using the `patch` command
on a compatible CKAN version.

# CKAN Open Data Customization
Mini project magang berupa pengembangan dan penyesuaian
portal Open Data Kabupaten Bangkalan berbasis CKAN.

## Lingkup Pekerjaan
- Penyesuaian antarmuka (UI)
- Pengaturan visibilitas dataset
- Optimalisasi navigasi pengguna
- Penambahan fitur permohonan data

## Teknologi
- CKAN
- Python
- HTML
- CSS
- Git

## Catatan
Perubahan dilakukan langsung pada core CKAN
tanpa menggunakan plugin ekstensi.

## Struktur Folder
- `patches/` : File patch yang mendokumentasikan modifikasi inti CKAN
- `docs/` : Penjelasan fitur dan tangkapan layar
- `ui/` : Referensi konfigurasi terkait UI

## Cara Menerapkan Patch (Opsional)
Patch dapat diterapkan secara manual atau menggunakan perintah `patch`
pada versi CKAN yang kompatibel.
